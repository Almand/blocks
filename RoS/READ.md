#Servidor Ruby, para compilar código del lenguaje 'C'

###Ejecutar
ruby basics.rb

###Abrir
Dentro del navegador web abrir:
[localhost](127.0.0.1:4567/form)

###Codear
Introducir código en C sin cabezera (#include<stdio.h>)
Ejemplo: 
`int main (void){
printf("Hola Mundo");
return 0;
}`

*Nota:* De momento solo funciona con la librería stdio
